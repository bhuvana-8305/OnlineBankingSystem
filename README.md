# Banking System in Java

A simple console-based Banking System project developed using Java without a database.

## Features
- User Registration
- User Login
- Deposit Money
- Withdraw Money
- Balance Inquiry

## Technologies Used
- Java
- VS Code

## How to Run

### Compile
```bash
javac BankingSystem.java
```

### Run
```bash
java BankingSystem
```

## Project Flow
Login/Register → Select Operation → Perform Transaction → Update Balance → Show Receipt
